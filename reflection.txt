Git в сучасних проєктах потрібен для контролю версій та командної роботи. Після виконання завдання я навчився створювати SSH-ключ та налаштовувати зв'язок з GitHub. Найзручніше — робота з гілками через GitHub web UI; найважче — спочатку розібратися з конфліктами та push/pull. Це допоможе у курсовому проєкті для синхронізації роботи групи.
